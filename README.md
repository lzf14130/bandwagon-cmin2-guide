## CMIN2 是什么，为什么移动用户要关心它

先说结论：**CMIN2（China Mobile International Next-generation Network）是中国移动自己的国际精品骨干网**，ASN 是 AS58807，对标电信的 CN2 GIA 地位。

很多人把 CMI 和 CMIN2 搞混。两个是同一家运营商，但完全不是一回事：
- AS58453 是普通 CMI，高峰期容易堵车，跟普通国际线路差别不大
- AS58807 才是 CMIN2，移动自己的精品骨干，带宽独立、不经过电信或联通的线路

判断方法很直接：路由追踪看 ASN，只有出现 58807 才算真正的 CMIN2。

搬瓦工 DC6/DC9 机房接入 CMIN2 之前，移动用户哪怕买了 CN2 GIA 机房，回程走的也是普通 CMI，效果跟电信用户的体验差了一截。现在这个问题解决了——移动用户到 DC6/DC9 的回程路由直接走 AS58807，实测延迟 150ms 左右，晚高峰丢包率极低。
